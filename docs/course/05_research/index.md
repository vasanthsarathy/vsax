# Module 5: Research and Extensions

Welcome to Module 5! This module prepares you for VSA research and extension development.

## Module Overview

**Learning Objectives:**
- Understand Vector Function Architecture (VFA)
- Design and implement custom encoders
- Explore current research frontiers and open problems
- Contribute to VSAX development

**Duration:** ~3-4 hours

**Prerequisites:** Modules 1-4

## Lessons

1. [Vector Function Architecture](01_vfa.md)
2. [Building Custom Encoders](02_custom_encoders.md)
3. [Research Frontiers and Open Problems](03_frontiers.md)

## Module Capstone

Design and prototype a research extension to VSAX for your domain.
